git branch -M main 
git remote add origin https://github.com/Net-bit/LR_1.git

… Или нажмите существующий репозиторий из командной строки

… Или импортировать код из другого репозитория
Вы можете инициализировать этот репозиторий кодом из проекта Subversion, Mercurial или TFS.

Контрольні запитання
1.	Що таке Git? 
Git – розподілена система керування версіями написана мовою С, командна оболонка на UNIX, Perl, Tcl, Python та C++.). З його допомогою можна відкотитися на більш стару версію вашого проекту, порівнювати, аналізувати, зливати зміни і багато іншого.

2.	В якому статусі можуть перебувати файли при роботі з Git? 
  a)Збережений у коміті означає, що дані безпечно збережено в локальній базі даних.

  b)Змінений означає, що у файл внесено редагування, які ще не збережено в базі даних.

  c)Індексований стан виникає тоді, коли ви позначаєте змінений файл у поточній версії, щоб ці зміни ввійшли до наступного знімку коміту.
  
3.	Для чого використовується  команда add? 
Команда git add додає вміст робочої директорії в індекс (staging area) для подальшого комміта.

4.	Для чого використовується  команда status? 
Команда git status показує стану файлів в робочій директорії і індексі: які файли змінені, але не додано в індекс; які очікують коммітов в індексі. Додатково до цього виводяться підказки про те, як змінити стан файлів.

5.	Для чого використовується  команда commit?
Команда git commit бере всі дані, додані в індекс за допомогою git add, і зберігає їх зліпок у внутрішній базі даних, а потім зрушує покажчик поточної гілки на цей зліпок.

6.	Для чого використовується  команда init? 
Ініціалізація Git у папці проекту.

7.	Що необхідно писати в коментарях комітів? 
Описуйте, що було змінено, де змінено, чому змінено.

8.	Як переглянути список комітів? 
Команда git log використовується для перегляду історії коммітов, починаючи з самого свіжого та йдучи до витоків проекту.

9.	Для чого використовується  команда checkout?
Команда git checkout використовується для перемикання гілок і вивантаження їх вмісту в робочу директорію.
